    Part Two: About.html

    Your Portfolio Website has been up for a couple weeks now but doesn't have an About Me Page to help people know more about you. Let's change that.

    1. Draw up a wireframe of an about page you think will represent you well. Don't let great be the enemy of good.

    MEANING: Don't spend to much time on the design or content but instead decide quickly and start coding. You're in a coding school not a design school. It can be a simple page!

    1. Your design must be for mobile, tablet, and laptop!

    2. You can use other websites as inspiration:

      • About 8-hour Day
      • About Kero1
      • About Nike

    3. Now go find your Portfolio Folder from Day 1 of this course, open it up and create a new .html file called "aboutPage" then create a .css file called "about-styles" to build this page in.

    4. Use all of the tools you've learned about so far and take this About Page to the next level.

    5. Remember to link the new page to your Portfolio's landing page.
    6. Remember your design must be designed for mobile, tablet, and laptop!
    7. git status add status commit push your work up to its repo.
    8. Then go to the URL that the website is hosted at and navigate to the "aboutPage" and copy/paste that URL to turn it in.
